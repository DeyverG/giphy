# Proyecto GIPHY CLON

Este proyecto fue creado para entender el funcionamiento de LazyLoad y custom Hooks con Typescript es una app que suministra Gif de la api de Giphy https://developers.giphy.com/

## Hooks Utilizados
  * UseState
  * UseEffect
  * UseRef
  * CustomHook

## Librerias Instaladas
  * Axios

## Famework UI
  * Ant Design
